# FACECORE 3D
A modern 3D arena shooter inspired by FaceBall 2000, built in Godot.

FACECORE 3D is an independent, original project that pays homage to retro FPS design while introducing modern multiplayer systems, VOIP, text chat, and fast-paced arena gameplay.  
This project is **not affiliated with**, **endorsed by**, or **sponsored by** any owners or creators of the original FaceBall titles.
